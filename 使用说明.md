# 故障报告邮件自动提醒系统使用说明

## 📋 系统概述

本系统基于 Google Apps Script 开发，实现故障报告的自动邮件提醒功能。系统能够根据车间和工序筛选需要提醒的邮箱，并按照设定的时间自动发送提醒邮件。

## 🚀 快速开始

### 1. 部署到 Google Apps Script

1. 打开 [Google Apps Script](https://script.google.com/)
2. 创建新项目
3. 将以下文件内容复制到项目中：
   - `故障报告邮件提醒.js` → 主程序文件
   - `定时执行.js` → 定时任务模块
4. 保存项目

### 2. 配置 Google Sheets

确保您的 Google Sheets 包含以下表：

#### Failure_Database 表（故障报告数据）
| 字段名 | 说明 | 示例 |
|--------|------|------|
| 编号 | 故障报告唯一标识 | FR001 |
| 机台号 | 故障发生的机台 | M001 |
| 问题描述 | 故障详细描述 | 设备无法启动 |
| 提交日期 | 故障提交时间 | 2024-01-15 |
| 车间 | 故障发生车间 | 车间A |
| 工序 | 故障发生工序 | 工序B |
| 故障报告编号 | 系统编号 | FR-2024-001 |
| 分配日期 | 任务分配时间 | 2024-01-15 |
| 上传日期 | 附件上传时间 | 2024-01-16 |
| 附件 | 相关附件 | 故障照片.jpg |

#### 通知清单表
| 车间 | 工序 | 邮箱 |
|------|------|------|
| 车间A | 工序B | user1@company.com |
| 车间A | 工序C | user2@company.com |
| 车间B | 工序A | user3@company.com |

### 3. 设置定时任务

在 Google Apps Script 编辑器中，运行以下函数之一：

- `quickSetup()` - 一键设置所有定时任务
- `setupAdvancedTiming()` - 高级定时配置
- `manualSetupTiming()` - 手动设置定时任务

## ⏰ 定时配置

### 默认时间设置
- **每周提醒**: 每周一上午 9:00
- **每日超期提醒**: 每天上午 8:30

### 自定义时间设置
如需修改执行时间，请编辑 `定时执行.js` 文件中的 `TIMING_CONFIG` 配置：

```javascript
const TIMING_CONFIG = {
  WEEKLY_REMINDER: {
    defaultTime: '09:00', // 修改为期望的时间
    defaultWeek: 1 // 1=周一, 2=周二, ..., 7=周日
  },
  DAILY_OVERDUE_REMINDER: {
    defaultTime: '08:30', // 修改为期望的时间
  }
};
```

## 📧 邮件功能

### 邮件类型

#### 1. 每周故障报告提醒
- **发送时间**: 每周一上午 9:00
- **内容**: 本周所有故障报告汇总
- **状态标识**: 正常/超期状态显示

#### 2. 每日超期故障报告提醒
- **发送时间**: 每天上午 8:30
- **内容**: 超过7天未处理的故障报告
- **紧急程度**: 红色警告样式

### 邮件筛选逻辑
- 基于 **车间** 和 **工序** 进行筛选
- 使用 **"与"** 的关系（车间A 且 工序B）
- 只向符合条件的邮箱发送相关故障报告

## 🔧 故障排除

### 常见问题

#### 1. 邮件发送失败
- 检查 Google Apps Script 权限设置
- 确认邮箱地址格式正确
- 查看执行日志中的错误信息

#### 2. 定时任务不执行
- 运行 `showCurrentTriggers()` 检查定时任务状态
- 确认时区设置正确（Asia/Shanghai）
- 检查 Google 服务状态

#### 3. 数据读取错误
- 确认 Google Sheets ID 正确
- 检查表名和字段名是否匹配
- 验证数据格式是否正确

### 调试方法

#### 1. 查看执行日志


#### 3. 手动执行测试

## 📝 自定义配置

### 修改超期天数阈值
在 `故障报告邮件提醒.js` 文件中修改：

```javascript
const CONFIG = {
  OVERDUE_DAYS: 7, // 修改为期望的天数
  // ... 其他配置
};
```

### 修改邮件模板
编辑 `generateWeeklyEmailContent()` 和 `generateOverdueEmailContent()` 函数来自定义邮件样式。

### 添加新的筛选条件
在 `filterFailureDataByWorkshopAndProcess()` 函数中添加新的筛选逻辑。
